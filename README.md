# Conecte Telecom Landing Page

Landing page moderna e responsiva desenvolvida para a Conecte Telecom, provedor regional de internet fibra óptica de Arroio do Sal - RS.

## 🚀 Objetivos

- Gerar leads
- Conversão via WhatsApp
- Fortalecer a marca
- Melhorar experiência mobile
- Destacar identidade regional

## 🛠 Tecnologias

- React
- Vite
- TailwindCSS
- Framer Motion
- shadcn/ui
- Lucide Icons

## ✨ Funcionalidades

- Layout responsivo
- Animações suaves
- Botão flutuante de WhatsApp
- Seção de planos
- FAQ
- Área institucional
- Design mobile-first
- Alta performance

## 📍 Sobre a Conecte

A Conecte Telecom é um provedor regional criado por moradores de Arroio do Sal, oferecendo internet 100% fibra óptica com atendimento humanizado e suporte local.
